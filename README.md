# Getting started

```bash
git clone https://github.com/thenaim/ngx-zenzen myApp
cd myApp
npm install
npm start
```

# Features

- Lazy-loading
- Auth pages (login, register, forgot and reset)
- Themes (default and dark)
- Ngrx
- LocalStorage
- PWA
- Responsive design
- Angular-material
- Bootstrap 4
